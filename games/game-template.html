Overview

Project Alpha is a high-stakes tactical shooter where players spawn into a hostile environment, gather loot, and attempt to extract. I am responsible for the core networking and moment-to-moment combat feel.


My Core Contributions

  • Built a custom networked weapon system using C++ with client-side prediction to ensure shooting feels responsive even on high ping.
  • Implemented the Gameplay Ability System (GAS) for player buffs, debuffs, and stamina management.
  • Designed a data-driven inventory using UDataAssets so designers can add new items without touching the code.

Technical Spotlight: Optimization

The Problem: When 20+ AI enemies spawned in the zone, server tick rates dropped drastically due to complex pathfinding and sight-checks.

The Solution: I refactored the AI perception system to operate on a time-sliced manager. Instead of every AI checking for the player every frame, the manager distributes the checks across multiple frames.
