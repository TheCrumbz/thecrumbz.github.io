UE5 Casino System

A fully functional casino backend built in UE5. This project focuses on secure logic, probability math, and dynamic UI feedback.

Key Features:

  • Probability-based "house edge" algorithms.
  • State machine for game phases (Betting, Spinning, Result).
  • Dynamic UI updates using CommonUI.

Technical Explanation

The core logic uses TMap to store payout tables and a custom RandomStream to ensure results are mathematically consistent. I handled the spin animation by decoupling the visual rotation from the actual result calculation to prevent lag from affecting the outcome.
